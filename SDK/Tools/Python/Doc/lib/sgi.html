16. SGI IRIX Specific Services

The modules described in this chapter provide interfaces to features that are unique to SGI's IRIX operating system (versions 4 and 5).

al Audio functions on the SGI.
AL Constants used with the \moduleal module.
cd Interface to the CD-ROM on Silicon Graphics systems.
fl FORMS library interface for GUI applications.
FL Constants used with the \modulefl module.
flp Functions for loading stored FORMS designs.
fm \emphFont Manager interface for SGI workstations.
gl Functions from the Silicon Graphics \emphGraphics Library.
DEVICE Constants used with the \modulegl module.
GL Constants used with the \modulegl module.
imgfile Support for SGI imglib files.
jpeg Read and write image files in compressed JPEG format.
