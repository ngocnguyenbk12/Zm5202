5.1 Information Units

Most of the environments should be described here: \excdesc, \funcdesc, etc.

\datadesc {name}
\datadesc {name}
Like \datadesc, but without creating any index entries.

\excdesc {name}
Describe an exception. This may be either a string exception or a class exception.

\funcdesc {name}{parameter list}
\funcdescni {name}{parameter list}
Like \funcdesc, but without creating any index entries.

\classdesc {name}{constructor parameter list}

\memberdesc {name}
\memberdescni {name}
Like \memberdesc, but without creating any index entries.

\methoddesc [class name]{name}{parameter list}
\methoddescni [class name]{name}{parameter list}
Like \methoddesc, but without creating any index entries.
